Methodology

Kept deliberately simple

This is not a scientific benchmark. It is a controlled creative test designed to show what each model builds from the same short brief.

  1. 01

    Empty folders

    Each model received a separate directory with no existing code or assets.

  2. 02

    Identical instructions

    The same prompt and medium reasoning effort were used for every run.

  3. 03

    Independent execution

    Each Codex CLI session selected, downloaded and integrated its own assets.

  4. 04

    Unedited results

    The published landing pages are the original outputs. No visual cleanup was applied.
